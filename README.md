# 🌍 Air Quality Assesment for Public health and management using IOT and Machine Learning

## 📌 Project Description
This project monitors air pollution levels using sensors and displays real-time data. 
It alerts users when pollution exceeds safe limits.

---

## ⚙️ Components Used
- Arduino uno
- Gas Sensors (simulated using potentiometers in Tinkercad)
- LCD Display (16x2)
- Buzzer
- Breadboard & Jumper Wires

---

## 🔧 Working Principle
- Sensors collect air quality data (CO, Gas, CO2 levels)
- Arduino reads sensor values
- Data is displayed on LCD and Serial Monitor
- If pollution level is high → buzzer turns ON
